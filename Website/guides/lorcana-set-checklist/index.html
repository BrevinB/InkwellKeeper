On top of the main sets, promo cards from events, organized play, and products form their own growing checklist — don't forget them when you're counting completion.

What "complete" means for a set

Most main sets run 204 cards (Attack of the Vine! pushes past that with 207), but that's just the base checklist. A true master set adds foils of every card plus the Enchanted alternate-art cards numbered above the set — see the Enchanted guide for how those work. Decide which completion you're chasing: base set, foil set, or master set.

Rotation: which sets are Core-legal

For competitive play, sets rotate: from July 24, 2026, Core Constructed uses Sets 9–13 (Fabled, Whispers in the Well, Winterspell, Wilds Unknown, Attack of the Vine!). Older sets remain legal in Infinity Constructed — full details in the deck building guide.
